# 房仲腳本生成器 v5｜多身分切換版

這是一個 React + Vite + Vercel API 的房仲短影音腳本生成器。

## v5 更新

- 新增多身分 Prompt Routing
- 不同輸出類型會切換不同 AI 身分：
  - 完整企劃：房仲短影音企劃顧問
  - 只要鉤子：爆款短影音開場鉤子設計師
  - 只要腳本：房仲短影音口播腳本大師
  - 只要貼文：專業社群文案人員
  - 時事口播推薦：房仲日更內容選題顧問
  - 藏鏡人模式：生活化雙人對談腳本大師

## 部署提醒

Vercel 需要設定環境變數：

```txt
OPENAI_API_KEY
```

## 本機測試

```bash
npm install
npm run dev
```
